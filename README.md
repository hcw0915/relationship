# 關係管理助手

一個幫助您管理關係的 React Native 手機應用程式，包含重要日期提醒、行動備忘錄、伴侶偏好記錄等功能。

## 功能特點

### 1. 重要日期 📅
- 記錄生日、紀念日等重要日期
- 設定提前幾天提醒
- 按日期排序顯示
- 支援新增、編輯、刪除

### 2. 行動備忘 ✓
- 建立待辦事項
- 設定優先級（低、中、高）
- 標記完成狀態
- 可設定截止日期
- 按狀態篩選（全部、待辦、已完成）

### 3. 伴侶偏好 ❤️
- 記錄伴侶的喜好、厭惡、過敏等資訊
- 分類管理（喜好、厭惡、過敏、飲食、興趣、習慣等）
- 支援標籤功能
- 按分類篩選

### 4. 個人資料 👤
- 用戶註冊與登入
- 設定伴侶名稱
- 查看使用統計

## 技術架構

- **前端框架**: React Native + Expo
- **導航**: React Navigation
- **資料庫**: Supabase (PostgreSQL)
- **身份驗證**: Supabase Auth
- **語言**: TypeScript

## 開發指南

### 安裝依賴

```bash
npm install
```

### 啟動應用

```bash
npm start
```

這將啟動 Expo 開發伺服器，您可以：
- 掃描 QR code 在真實手機上測試（需要 Expo Go app）
- 按 `a` 在 Android 模擬器中開啟
- 按 `i` 在 iOS 模擬器中開啟
- 按 `w` 在瀏覽器中開啟

### 環境變數

應用程式需要以下環境變數（已在 `.env` 中配置）：
- `EXPO_PUBLIC_SUPABASE_URL`: Supabase 專案 URL
- `EXPO_PUBLIC_SUPABASE_ANON_KEY`: Supabase 匿名金鑰

## 資料庫結構

應用程式使用 4 個主要資料表：

1. **profiles** - 用戶個人資料
2. **important_dates** - 重要日期
3. **action_memos** - 行動備忘錄
4. **partner_preferences** - 伴侶偏好

所有資料表都啟用了 Row Level Security (RLS)，確保用戶只能存取自己的資料。

## 使用說明

1. 首次使用請先註冊帳號
2. 登入後可以開始使用各項功能
3. 點擊右下角的「+」按鈕新增項目
4. 點擊項目可以編輯
5. 長按項目可以刪除

## 注意事項

- 本地開發優先，部署功能暫不需要
- 所有資料儲存在 Supabase 雲端資料庫
- 需要網路連線才能正常使用
